# GitHub Actions CI/CD Example

This repository demonstrates a simple GitHub Actions CI workflow for a Node.js project.

## What is included

- `src/math.js` - a small function
- `tests/math.test.js` - Jest unit test
- `.github/workflows/ci.yml` - workflow that installs dependencies, runs tests, and starts the app

## How to use

1. Commit and push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Actions` and watch the `CI` workflow run on your push.

## Commands

- `npm install`
- `npm test`
- `npm start`
